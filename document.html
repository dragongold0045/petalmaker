Interface Overview

Know where everything lives before you start
01

The editor is divided into five main zones:

A
Header Bar (Top)
Contains the shape key input, Undo/Redo, Theme toggle, Canvas clear, and global actions like Import/Export JSON and Code Export.
B
Context Bar (Below Header)
Provides quick access to your current tool state, Smart Snap settings, Grid/Handles toggles, and viewport Zoom/Fit controls.
C
Left Panel — Tools & Settings
Houses all drawing tools, Presets, the new 🤖 AI Trace feature, Canvas View options, Smart Snap rules, and the Reference Image uploader.
D
Canvas (Center)
The main interactive area. Use your mouse to draw paths, drag points, or freehand sketch.
E
Right Panel — Layers, Props & Export
Manage your stack in Layers, adjust selected node data and styles in Props, or preview the generated code in Export and JSON tabs.
🔭

Navigating & Context Bar

Pan freely and utilize the new Context Bar
02
1
Zoom & Pan
Roll the mouse wheel to zoom at your cursor. Hold Middle Click or press H to enter Pan mode and drag the canvas.
2
Context Bar Zoom Controls
Located just below the main header, the Context Bar offers − / + buttons for zooming. Click ⊡ Fit or the percentage label to instantly reset the view to 100%.
3
Quick Toggles
The Context Bar also allows you to toggle ⊞ Grid, ◎ Handles, and 🧲 Snap without digging into the side panels.

Drawing Tools

Vector precision, freehand sketching, and typography
03
↖
Select / Drag 1
Move anchor points and curve handles. Switch back to this after placing points to adjust them.
╱
Line Point 2
Places a sharp corner L segment for geometric shapes.
⌒
Quad Curve 3
Places a quadratic bezier Q (one control handle). Good for simple arcs.
〜
Cubic Curve 4
Places a cubic bezier C (two handles). Best for organic, smooth shapes.
⌇
Subpath 5
Starts a M moveTo. Lifts the pen to draw a disconnected region inside the same layer.
✏
Freehand 6
Sketch freely. Points are automatically simplified upon release. Requires a Stroke layer.
▭
Shapes 7, 8, 9
Drag to draw an Arc, Rectangle, or Ellipse. Instantly generates editable bezier points.
T
Text Tool T
Adds typography. Requires a Text layer. Move the text anchor via the Canvas, edit content in Props.

Smart Snap

Achieve pixel-perfect alignment effortlessly
04

Version 6 introduces a robust snapping engine. Access it in the Left Panel under Smart Snap or toggle it via the Context Bar (S).

1
⊕ Axis Snapping
Automatically aligns your points to the exact X=0 or Y=0 centerlines of the canvas origin.
2
⊞ Grid Snapping
Snaps to mathematical grid intervals. Adjust the grid resolution in the Context Bar dropdown (0.05, 0.1, 0.25, 0.5).
3
◉ Point Snapping
Snaps your cursor exactly to the coordinates of existing nodes in other layers. Great for closing gaps between separate shapes.
4
Snap Threshold
Adjust the Threshold slider to determine how "magnetic" the snapping feels. A lower value requires your mouse to be closer before snapping occurs.
✨

AI Trace (Image → Shape)

Automatically convert transparent images into vector layers
05
🖼Drop Image
⚙️Tweak Params
🤖Trace
✅Export Vector

Found in the Left Panel, AI Trace uses an advanced contour-finding algorithm to vectorise images directly in the browser.

1
Drop an Image
Upload a PNG or WebP with a transparent background into the AI drop zone. Click ✨ Trace to open the modal.
2
Adjust Alpha & Smoothing
In the modal, use Alpha threshold to dictate what pixels are considered "solid". Increase Smooth passes to remove jagged pixel edges before tracing.
3
Simplification
The Simplify slider uses Ramer-Douglas-Peucker to reduce point count. A higher value yields fewer points but may lose sharp details.
4
Output Modes
Select how the AI constructs layers: Single fill, Fill + Outline, Shadow + Fill, or Full (Shadow + Fill + Outline) for an instant stylized composition.
📑

Layers & Properties

Organize, duplicate, and style everything in the Right Panel
06
1
Layer Types & Reordering
Add Path, Circle, Stroke, or the new Text layer. Grab the ⠿ handle on any row to drag-and-drop its rendering order (top = foreground).
2
Properties Panel
Select a layer to view its properties. Edit colors, toggles, offsets, rotation, and radial Repeat counts.
3
Text Layer Properties
If a Text layer is selected, use the Props panel to type the Content, set Font size, choose a FontFamily, adjust Weight (Normal/Bold/900), and alignment.
4
Stroke Styling
Every layer supports Strokes. Set Color, Width, Line Cap (Butt/Round/Square), Join (Miter/Round/Bevel), and Dash patterns (Solid, Dashed, Dotted, Dash-dot).
🖼

Reference Image

Manually trace artwork with an overlay
07
1
Upload & Scale
Drop an image into the Reference Image zone (Left Panel). Adjust the Opacity to ~30% and scale it to match your desired canvas size.
2
Move Ref Tool (R)
Select the Ref tool (or press R). A green crosshair anchor will appear on the image; drag it to offset the reference.
3
Center & Remove
Use the ⊕ Center button to instantly snap the reference back to the (0,0) origin. Click ✕ Remove to discard it.

Export (Code & JSON)

Multi-platform code generation and native saving
09
1
Multi-Target Code Export
In the Right Panel's Export tab, choose your target: PixiJS (TypeScript graphics), Canvas 2D (Native JS rendering), or WebGL (GLSL Shader using Signed Distance Fields).
2
Preview Canvas
The Export tab features a mini-preview window showing exactly how your chosen engine (Pixi/Canvas/WebGL) will render the math in real-time.
3
JSON Export & Import
Since Undo/Redo history resets on refresh, use the JSON tab (or the 📥/📤 buttons in the header) to download your full project state. Import the `.json` file later to resume exactly where you left off.

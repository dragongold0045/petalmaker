Interface Overview

Know where everything lives before you start
01

The editor is divided into four zones:

A
Header Bar (top)
Contains the shape key name input, Undo / Redo buttons, the 🌙 Dark/Light mode toggle, Clear, Copy, and Export .ts.
B
Left Panel — Tools & Settings
All drawing tools (keys 1–9), preset templates, canvas display options (grid, handles, repeats, snap), and the Reference Image uploader.
C
Canvas (center)
The main interactive area. Scroll to zoom, drag with Middle Click or the Pan tool (H) to move around. A checkerboard background and optional grid help you see spacing clearly.
D
Right Panel — Layers & Code
Switch between the Layers tab (manage and style every layer) and the Code tab (live-generated TypeScript for PixiJS). Use drag-and-drop to reorder layers.
E
Status Bar (bottom)
Shows the active layer name, current tool, point count, and zoom level. Click the zoom percentage to snap back to 100%.
🔭

Navigating the Canvas

Pan freely and zoom to any level
02
1
Zoom with the scroll wheel
Roll your mouse wheel up to zoom in and down to zoom out. The zoom centers on your cursor position — just like Figma or Photoshop. Range: 8 % → 800 %.
2
Pan with Middle Click or the Pan tool
Hold Middle Click and drag to pan at any time. Alternatively, press H (or click the ✋ Pan button in the left panel) to enter Pan mode, then drag with the left mouse button.
3
Zoom controls in the status bar
Click − or + buttons in the bottom bar for step-by-step zoom. Click the 100% percentage label to instantly reset the view to default position and zoom.
4
Keyboard zoom shortcuts
Press = or + to zoom in, - to zoom out, and 0 to reset view — all without touching the mouse.

The size ring (dashed blue circle) always shows the shape's actual output pixel size. The number next to it corresponds to the Preview size slider in the Canvas settings section.

Your First Shape

Draw a path from scratch in under a minute
03
＋Add Layer
✏Pick Tool
🖱Click Canvas
🎨Style It
⬇Export
1
Add a layer
In the Layers tab on the right, click + Path to create a vector path, + Circle for a circle, or + Stroke for a freehand stroke layer.
2
Select a drawing tool
In the left panel under Tools, pick Line pt (2) to place sharp corners, or Cubic C (4) for smooth bezier curves. The active tool is highlighted in blue.
3
Click on the canvas to place points
Each click drops a new point. For curve tools, two control handles appear automatically — you can drag them later to adjust the curve shape.
4
Drag points and handles to refine
Switch back to Select (1) and drag any anchor point (blue dot) or curve handle (yellow / purple diamond) directly on the canvas. The orange crosshair at the center is the layer's offset origin — drag it to shift the whole layer.
5
Adjust the fill color and transform
With the layer selected in the right panel, change Fill color, adjust Scale, Rotation, Offset X/Y, and set Repeat to tile the shape radially (e.g., 5 petals around a center).
6
Close or leave the path open
Toggle Close path in the layer properties to automatically connect the last point back to the first. Turn it off for open curves like arcs or strokes.

Delete a point: Select it (click its dot on the canvas or click its row in the Points list), then press Del. The path stays intact if there are at least 2 remaining points.

Drawing Tools

Every tool explained, with when to use each one
04
↖
Select / Drag 1
Move existing anchor points and curve handles. The default mode — switch back here after placing points.
✋
Pan H
Click and drag the canvas to move the viewport. Works on top of any shape without accidentally selecting points.
╱
Line Point 2
Places a sharp corner L segment. Best for polygons, diamonds, arrows, and any shape with hard edges.
⌒
Quad Curve 3
Places a quadratic bezier Q — one control handle. Good for simple gentle arcs with less complexity than cubic.
〜
Cubic Curve 4
Places a cubic bezier C — two independent handles. The most powerful curve; use it for smooth, organic outlines like petals and blobs.
⌇
New Subpath 5
Starts a M moveTo — lifts the pen so you can draw a separate disconnected region inside the same path layer (e.g., a donut hole).
✏
Freehand 6
Hold and drag to draw freely. Points are recorded as you move and automatically simplified on release (Ramer–Douglas–Peucker). Works on Stroke layers only.
◠
Arc 7
Drag to set a bounding box, then releases a half-ellipse arc. Works on Path layers — useful as a quick rounded cap or decorative curve.
▭
Rectangle 8
Drag a bounding box and release — instantly converts to 4 line-segment path points. Great starting point for badges, frames, or UI elements.
⬭
Ellipse 9
Drag to draw and release — converted to a 4-point cubic bezier ellipse (fully editable). More flexible than a Circle layer as each handle can be individually shaped.

Arc, Rectangle, and Ellipse require an active Path layer. They replace the existing points on that layer, so add a new path first if you want to keep your current shape.

Adjusting curve handles after placement: Switch to Select (1), then drag the yellow handle (h1 — for Q and C curves) or the purple handle (h2 — C curves only). You can also type exact values in the node list in the right panel.

Layers Panel

Organize, reorder, and style multiple shapes
05
1
Selecting a layer
Click any row in the layer list to make it active (highlighted in blue). Its handles appear on the canvas and its properties load in the panel below.
2
Reordering with drag-and-drop
Grab the ⠿ handle on the left of any layer row and drag it up or down to change the render order. Layers at the top of the list are drawn on top of layers below them.
3
Toggling visibility
Click the 👁 eye icon to hide or show a layer without deleting it. Hidden layers are excluded from the exported code.
4
Duplicating a layer
Click the ⧉ icon to create an exact copy. The copy is inserted directly above the original and becomes the active layer immediately.
5
Radial repeat
In the layer properties, drag the Repeat slider from 1 to 16. The shape is cloned and evenly rotated around the center point — perfect for petals, blades, or gear teeth. The Rotation slider sets the starting angle of the first copy.
6
Layer types
path — vector shape with bezier nodes   circle — a perfect circle with radius control   stroke — a freehand line with stroke styling only (no fill)

Stroke Styles

Add outlines and line styles to any layer
06

Every Path, Circle, and Stroke layer has its own Stroke section in the properties panel. Scroll down in the right panel after selecting a layer to find it.

1
Enable stroke
Toggle Enable stroke on. For Path and Circle layers, the stroke is drawn on top of the fill. Stroke layers have it enabled by default (they have no fill).
2
Set stroke color and width
Click the Stroke color swatch to open the color picker. Drag the Width slider — values are in normalized units (0.03 ≈ 3% of the shape size, which is a medium-weight line at 140px preview).
3
Choose a dash pattern
The Dash dropdown offers four options:
• Solid — continuous line (default)
• Dashed — long gaps between dashes
• Dotted — tight dots
• Dash-dot — alternating dash and dot
4
Line Cap & Line Join
Cap controls how line ends look: Butt (flat cut), Round (semicircle), Square (flat with overhang).
Join controls corners: Miter (sharp spike), Round (rounded), Bevel (clipped corner). For freehand strokes, Round cap + Round join gives the smoothest look.

Pro tip: Duplicate a shape layer and set the copy to a slightly larger scale with a contrasting stroke color. This creates a clean outline / border effect on top of your base fill.

Reference Image

Trace any artwork with a semi-transparent overlay
07
1
Upload an image
In the left panel under Reference Image, click the dashed zone or drag-and-drop any PNG, JPG, SVG, or WebP file onto it. The image appears centered over the canvas immediately.
2
Adjust opacity
Drag the Opacity slider. A value around 25–40% is ideal for tracing — the image is visible but your shapes stay readable on top.
3
Scale the reference
Use the Scale slider to resize the reference image to match your shape's bounding box. The image stays centered on the canvas origin.
4
Toggle visibility
Use the Visible checkbox to hide the reference without removing it. This lets you quickly compare your shape against the original.
5
Remove the image
Click ✕ Remove image to clear it. The image is only stored in your current browser session — it is never uploaded to any server.

The reference image is a visual guide only — it does not appear in the exported code or affect the shape output in any way.

Undo & Redo

80-step history with keyboard shortcuts
08
1
Undo the last action
Press Ctrl + Z (or ⌘ + Z on Mac), or click ↩ Undo in the header bar. Steps back through shape edits, point changes, layer adds/deletes, and reorders.
2
Redo an undone action
Press Ctrl + Shift + Z or Ctrl + Y, or click ↪ Redo. Both the Undo and Redo buttons are greyed out when there is nothing left in that direction.
3
What gets recorded
A history snapshot is saved whenever you: add or delete a layer, add or delete a point, commit a drag operation (on mouseup), change segment type, reorder layers, load a preset, or clear the canvas. Slider drags mid-motion do not create intermediate snapshots.

History is session-only — it resets when you reload the page. Export your work regularly with Export .ts to preserve it.

Exporting Code

Get production-ready TypeScript for PixiJS
10
1
Set the shape key
In the header bar, edit the Shape key field (e.g., PETALS.MY_FLOWER). This becomes the object key in the generated code, so name it to match your project's naming convention.
2
Preview in the Code tab
Switch to the Code tab in the right panel to see the generated TypeScript update in real time as you draw. Syntax is color-highlighted for readability.
3
Copy to clipboard
Click ⎘ Copy in the header or the Copy button in the Code tab header to copy the full code block.
4
Download as a .ts file
Click ↓ Export .ts to save a TypeScript file named after your shape key (e.g., PETALS_MY_FLOWER.ts). Drop this file straight into your PixiJS project.

The generated code uses a rot2d helper for rotation math, then calls PixiJS Graphics methods: beginFill, moveTo, lineTo, quadraticCurveTo, bezierCurveTo, closePath, endFill, lineStyle, and drawCircle. Hidden layers are automatically excluded.

Tips & Best Practices

Faster workflows and better results
12
✦
Use "Shadow → Outline → Fill" layer order
For depth, create three copies of the same path: the bottom one slightly offset and darkened (shadow), the middle scaled slightly larger (outline), and the top at normal scale (fill). This is the technique used by all built-in presets.
✦
Enable "Snap to 0.05" for clean coordinates
Turn on Snap to 0.05 in the Canvas settings while placing points. This produces clean numbers in the exported code and prevents sub-pixel jitter in the final render.
✦
Design at a small size, test at large
Keep the Preview size slider around 120–160px while editing. Before exporting, drag it up to 200px+ to check that handles and curves look right at a larger scale.
✦
Hide layers to focus
Toggle the 👁 eye icon on layers you're not currently editing. Complex shapes with 6+ layers can get visually noisy — working with just one layer visible at a time makes precise edits much easier.
✦
Fine-tune numbers in the points list
The node list in the right panel shows exact coordinate inputs for every point and handle. Click any number field and type a precise value — much more accurate than dragging for small adjustments.
✦
Freehand strokes work best zoomed in
Zoom in to 200–300% before using the Freehand tool. A larger canvas area gives you more pixel resolution per stroke, which results in smoother simplified curves after release.
✦
Export often
There is no auto-save. Your work lives only in the current browser tab. Click Export .ts any time you reach a state you want to keep — the file is tiny and reopens in your code editor instantly.
